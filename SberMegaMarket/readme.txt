Скрипт для поиска товаров с максимальными бонусами.

Использование:
1. клонировать репозиторий
2. перейти в корневую папку проекта используя GitBash
3. установить необходимые библиотеки из файла requirements.txt
gitbash>> pip install requirements.txt
4. открыть в редакторе файл main
5. в ячейке variables изменить переменные, используя подсказки
6. запустить все ячейки на выполнение.

В результате, ниже ячейки variables будет доступен датафрейм df с таблицей по всем товарам из указанного каталога.
В конце файла реализован поиск лучшего предложения с выводом ссылки на товар.

Для запуска телеграмм бота:
1. клонируйте репозиторий
2. в GitBash из папки репозитория выполните:
pip install requirements.txt
3. откройте в редакторе файл tg_bot_v1.py
4. Укажите в переменных необходимые значения:
tg_token - ключ к Вашему боту;
city - город доставки
max_page - количество просматриваемых страниц (не больше 40).
5. запустите файл tg_bot_v1.py из редактора, либо командной строки.

При отправке боту ссылки на каталог с товарами, бот просмотрит указанное в переменной max_page число страниц каталога и выдаст ссылки на 3 варианта товаров, с максимальным процентом возврата в виде бонусов.
