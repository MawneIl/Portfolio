Цель проекта: обучить модель, предсказывающую целевое действие, на основе предоставленных данных о визитах на сайт сервиса "СберАвтоподписка".
Для удобства получения предсказаний, модель обернута в localhost web app, с помощью фреймворка FastApi с валидацией данных.

Запуск приложения:
- Скопируйте репозиторий проекта
- Откройте терминал Bash и перейдите в корневой каталог проекта
- Установите необходимые библиотеки, выполнив команду 'pip install requirements.txt'
- Запустите приложение командой 'uvicorn main:app --reload'

Запросы:
- Откройте Postman
- Выполните один из запросов:
1. Проверить готовность приложения:
GET>> http://127.0.0.1:8000/status
2. Узнать версию приложения и метаданные:
GET>> http://127.0.0.1:8000/version
3. Получить предсказание модели:
POST>> http://127.0.0.1:8000/predict
* передайте один из json'ов из папки 'Samples'. Для этого выберете 'Body -> raw -> JSON -> вставьте данные в поле.
